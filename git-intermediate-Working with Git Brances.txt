CHERRY PICKING
--------------

- Copy specific commits to another branch
	git cherry-pick <commit>

- Possible uses:
	Bugfix for multiple versions of product
	Capture commits from inactive branch


- Check commits from other branch:
	git log <branch-name> --oneline
